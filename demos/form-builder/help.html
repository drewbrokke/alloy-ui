Key Function
f Focus on fields
Arrow Keys Move Selection
j or down arrow Swap field position down
k or up arrow Swap field position up
Right arrow (If field has a child) Select child
Left Arrow (If field has a parent) Select parent
n Create new field
c Create child field
d Duplicate field
Enter Begin/Stop Editing
Delete or Backspace Delete field
e Edit field properties
Number Keys (During pop-up selection) Highlight Option
Number Keys (press twice) (During pop-up selection) Highlight and Select Option
